# 热工设备多层绝热结构温度计算程序

## 功能特性

本程序用于计算热工设备中高温烟气通过多层绝热结构时的温度分布,包括:

### 核心功能

1. **多层导热计算**
   - 支持任意数量的材料层
   - 计算每层的热面和冷面温度
   - 考虑温度依赖的导热系数(二项式模型)

2. **温度依赖导热系数**
   - 导热系数模型: `k(T) = k0 * (1 + a*T + b*T^2)`
   - 高精度的平均导热系数计算
   - 支持高温范围(300K ~ 1500K)

3. **对流换热分析**
   - 烟气侧对流: 基于Churchill-Bernstein相关式
   - 环境侧对流: 考虑风速和设备几何
   - 自动计算对流系数,无需手动输入

4. **环境边界条件**
   - 结合风冷对流
   - 表面辐射(Stefan-Boltzmann方程)
   - 等效线性化辐射系数

5. **详细的可视化**
   - 温度-厚度曲线(阶梯型分布)
   - 各层温度对比(热面vs冷面)
   - 温度降分布
   - 热阻分析(饼图和柱状图)
   - 敏感性分析结果

6. **敏感性分析**
   - 风速对外壳温度的影响
   - 烟气温度对温度分布的影响
   - 环境温度对外壳温度的影响
   - 材料参数变化的影响
